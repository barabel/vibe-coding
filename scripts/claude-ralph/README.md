# Ralph Loop

## Как запустить

**Подготовка (один раз):**

1. Открой **Git Bash** — правой кнопкой в папке проекта → *Git Bash Here*, или в VS Code выбери Git Bash как терминал
2. Убедись что `claude` доступен: `claude --version`
3. Убедись что находишься в корне проекта: `pwd` должен показать путь к проекту

---

**Каждая итерация:**

1. Запусти скрипт:
   ```sh
   bash scripts/claude-ralph/loop.sh refactor-use-view-items-animations
   ```
2. Claude откроется в интерактивном режиме в том же терминале
3. Наблюдай — он читает PRD и progress, находит следующую issue, реализует
4. Когда Claude просит подтвердить shell-команду (git, tsc и т.д.) — ты видишь запрос, одобряешь или отклоняешь
5. По завершении Claude сам закрывает сессию
6. Проверь что сделано: `git log --oneline -3` и посмотри `docs/progress/`

---

**Следующая итерация:**

7. Если всё ок — снова запускаешь шаг 1
8. Повторяешь до тех пор, пока Claude не ответит `<promise>COMPLETE</promise>`

---

**Если что-то пошло не так:**

- Отменить текущую сессию — `Ctrl+C`
- Откатить коммит — `git revert HEAD` или `git reset --soft HEAD~1`
- Исправить вручную и запустить снова
